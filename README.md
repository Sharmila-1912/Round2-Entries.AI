# Problem 1 - Second Largest Distinct Element

### Approach
The program reads all elements from the array and finds the largest and second largest distinct values using variables.  
Duplicates are ignored while finding the second largest element.  
If no second distinct value exists, the program prints -1.

### Run Command
```command Prompt
javac Problem1.java
java Problem1
```

# Problem 2 - Palindrome Checker

### Approach
The input string is converted to lowercase and all non-alphanumeric characters are removed.  
The cleaned string is reversed using StringBuilder and compared with the original cleaned string.  
If both are equal, the string is a palindrome.

### Run Command
```Command Prompt
javac Problem2.java
java Problem2
```

# Problem 3 - Most Frequent Word
### Approach
The sentence is converted to lowercase and split into words.  
Nested loops are used to count the frequency of each word.  
If two words have the same frequency, the alphabetically smaller word is selected.
### Run Command
```Command Prompt
javac Problem3.java
java Problem3
